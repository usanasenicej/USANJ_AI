## ⚡ Quick Start Deployment

Initialize the repository locally to access the secure USANJ node:

### 1. Clone the Protocol
```bash
git clone https://github.com/yourusername/usanj-ai-dashboard.git
cd usanj-ai-dashboard
```

### 2. Synthesize Dependencies
```bash
npm install
# or
yarn install
# or
pnpm install
```

### 3. Ignite the Local Node
```bash
npm run dev
# The architectural grid will automatically host at http://localhost:5173
```
